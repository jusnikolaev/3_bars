# Ближайшие бары

Программа, позволяющая найти самый большой/маленький/ближайший бар. 
Список баров скачен с data.mos.ru

# Как запустить

Скрипт требует для своей работы установленного интерпретатора Python версии 3.5

Запуск на Linux:

$ python3 bars.py
